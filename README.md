# Work Tracker

## Setup Instructions

### Set up environment variables
`cp env.sample .env`
 Update `.env` with a more secure password

### Build docker images
`docker compose build`

## Running instructions

`docker compose up`

### Start the API

## Running

Open in browser: `localhost:8080`
